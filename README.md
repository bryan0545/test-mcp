# test-mcp

Este es un repositorio de prueba para experimentar con MCP (Model Context Protocol). 

## Descripción

Este proyecto está diseñado para realizar tests y pruebas de funcionalidad relacionadas con MCP, permitiendo explorar las capacidades y características del protocolo de contexto de modelos.

## Propósito

- Experimentar con las funcionalidades de MCP
- Realizar pruebas de integración
- Validar diferentes casos de uso
- Documentar aprendizajes y descubrimientos
